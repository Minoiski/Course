\section{Построение математической модели и формулировка задачи оптимального управления}\label{2sec:problem-formulation}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
Неоклассическая модель оптимального экономического роста [5] описывает замкнутую агрегированную экономику, производящую в каждый момент времении $t \geq 0$  единственныйоднородный продукт (капитал)со скоростью $Y(t) > 0$ В каждыймомент времени $t$ величина $Y(t)$ является функцией текущих значений капитала $K(t) > 0$ и трудовых ресурсов $L(t) > 0$; трудовые ресурсы также предполагаются однородными. Таким образом,

\begin{equation}\label{2.1}
  Y (t)=F(K(t),L(t)) \text{ для любого } t \geq 0.
\end{equation}

Функция $F$ обычно называется производственной функцией. Относительно производственной функции $F$ предполагается, что она определена и непрерывна на положительном квадранте
$$G=\{ (K,L)\in \mathbb{R}^2 \colon K>0,L>0\},$$
дважды непрерывно дифференцируема и удовлетворяет следующим “неоклассическим” условиям для всех $K>0$, $L>0$:

\begin{equation}\label{2.2}
  \frac{\partial F(K,L)}{\partial K}>0, \hspace{1cm} \frac{\partial^2 F(K,L)}{\partial K^2}<0,
\end{equation}
\begin{equation}\label{2.3}
  \frac{\partial F(K,L)}{\partial L}>0, \hspace{1cm} \frac{\partial^2 F(K,L)}{\partial L^2}<0,
\end{equation}
\begin{equation}\label{2.4}
  \lim\limits_{K \to +0} \frac{\partial F(K,L)}{\partial K}=\infty, \hspace{1cm} \lim\limits_{K \to \infty} \frac{\partial F(K,L)}{\partial K}=0,
\end{equation}
\begin{equation}\label{2.5}
  \lim\limits_{K \to +0} \frac{\partial F(K,L)}{\partial L}=\infty, \hspace{1cm} \lim\limits_{K \to \infty} \frac{\partial F(K,L)}{\partial L}=0.
\end{equation}

Наконец, предполагается, что $F$ положительно однородна первой степени, т.е.
\begin{equation}\label{2.6}
  F(\lambda K, \lambda L)= \lambda F(K,L)\text{ для любых } \lambda>0, K>0, L>0.
\end{equation}

Последнее условие означает, что объем производства в каждую единицу времени прямо пропорционален величинам имеющихся вэту единицу времени производственных факторов. В качестве производственной функции $F$ может фигурировать, например, стандартная функция Кобба–Дугласа вида
\begin{center}
  $F(K,L)=AK^{\alpha_1}L^{\alpha_2}$,
\end{center}
где $A>0, \alpha_1>0, \alpha_2>0$ и $\alpha_1+\alpha_2=1.$

В замкнутой экономике произведенный продукт либо инвестируется в основные производственные фонды (капитал), либо потребляется. Предположим, что в каждый момент времени $t \ge 0$ минимально возможная часть потребляемого продукта есть  $\epsilon Y(t) > 0$, где $0<\epsilon<1$ --- некоторая постоянная, а доля продукта $(1 - \epsilon)Y(t)$ может быть распределена между производством и потреблением произвольным образом.

 Пусть в момент времени $t \ge 0$ часть

\begin{equation}\label{2.7}
  I(t)=u(t)Y(t), 0\le u(t)\le 1 - \epsilon,
\end{equation}
произведенного продукта инвестируется в основные производственные фонды, а оставшаяся часть

\begin{equation}\label{2.8}
  C(t)=(1-u(t))Y(t)
\end{equation}
потребляется. В дальнейшем величина $u(t) \in [ 0,1 - \epsilon ]$ будет трактоваться как значение управления в момент времени $t$.

В данной модели амортизации капитала не предполагается. Поэтому в силу равенства (\ref{2.7}) динамика изменения капитала может быть описана при помощи следующего дифференциального уравнения:

\begin{equation}\label{2.9}
  \dot K(t)=I(t)=u(t)Y(t).
\end{equation}

Считаем, что в начальный момент времени $K(0)=K_0>0$.

Пусть трудовые ресурсы удовлетворяют условию экспоненциального роста, т.е.

\begin{equation}\label{2.10}
  \dot L(t)=\mu L(t),
\end{equation}
где $\mu>0$ ---  некоторая постоянная. Аналогично будем считать, что $L(0)=L_0>0$.

Пусть $\rho > 0$ ---  параметр дисконтирования и в каждый момент времени $t\ge 0$  мгновенная полезность  g(K(t),L(t),u(t)) текущего процесса управления есть логарифм полного потребления C(t), т.е. (см. (\ref{2.1}), (\ref{2.8}))
\begin{center}
  $g(K(t),L(t),u(t)) = lnC(t) = ln(1-u(t))+lnF(K(t),L(t)). $
\end{center}

Неоклассическая модель оптимального экономического роста (c логарифмической функцией мгновенной полезности) формулируется в виде следующей задачи оптимального управления $(P_\epsilon), 0<\epsilon<1$:

\begin{center}
$\dot K(t)=u(t)F(K(t),L(t)), u(t)\in U_{\epsilon}=[0, 1-\epsilon ],$
\end{center}

\begin{center}
  $\dot L(t)=\mu L(t),$
$K(0)=K_0, L(0)=L_0,$
\end{center}
\begin{equation}\label{2.11}
  J(K,L,u)=\int_{0}^{\infty}e^{-pt} [\ln(1-u(t))+lnF(K(t),L(t)) ]dt \to \max.
\end{equation}

Задача $(P_{\epsilon})$ является частным случаем задачи $(P)$. При исследовании неоклассической задачи оптимального экономического роста обычно, используя условие однородности (\ref{2.6}), понижают размерность системы и переходят к вспомогательной фазовой переменной $x = K$/$L$ (величине капитала, приходящегося на единицу рабочей силы) и однофакторной производственной функции $f$ вида $f(x)=F(x,1), x>0$. В этом случае в силу условий (\ref{2.1}) и (\ref{2.6}) для любого $t \ge 0$

  $$\frac{Y(t)}{L(t)}=\frac{1}{L(t)}F(K(t),L(t))=F(\frac{K(t)}{L(t)},1)=f(x(t)).$$

Функция $f$ определена и непрерывна на $\tilde G=(0, \infty)$. В силу условий (\ref{2.2}) для всех $x>0$
\begin{equation}\label{2.12}
  \frac{d}{dx}f(x)>0, \frac{d^2}{dx^2}f(x)<0
\end{equation}
и вследствие (\ref{2.2})–(\ref{2.6})
\begin{equation}\label{2.13}
  \lim_{x \to +0}f(x)=0, \lim_{x \to \infty}f(x)=\infty, \lim_{x \to +0} \frac{d}{dx}f(x)=\infty, \lim_{x \to \infty} \frac{d}{dx}f(x)=0.
\end{equation}

Для переменной x(t)=K(t)/L(t) в силу равенств (\ref{2.9}) и (\ref{2.10}) имеем

  $$\dot x(t)=\frac{d}{dt} \frac{K(t)}{L(t)}=\dot K(t) \frac{1}{L(t)}-\frac{K(t)}{L^2(t)}\dot L(t)=u(t) \frac{Y(t)}{L(t)}-\mu \frac{K(t)}{L(t)},$$

откуда в силу определения переменной x и условий (\ref{2.1}) и (\ref{2.6}) вытекает равенство

  $$\dot x(t)=u(t)f(x(t)-\mu x(t)).$$


Величина мгновенного потребления на единицу трудовых ресурсов в момент времени $t \ge 0$ есть c(t)=C(t)/L(t). Согласно (\ref{2.1}) и (\ref{2.8}) получаем

  $$c(t)=(1-u(t))\frac{Y(t)}{L(t)} = (1-u(t))f(x(t)).$$


Заметим, что в силу равенства (\ref{2.10}) трудовые ресурсы $L$ в рассматриваемой модели подчиняются заранее заданной динамике. Поэтому максимизация интегрального функционала (\ref{2.11}) эквивалентна задаче максимизации функционала
\begin{center}
  $J(x,u)=\int_{0}^{\infty}e^{-pt}\ln c(t)dt=\int_{0}^{\infty}e^{-pt} [  \ln (1-u(t))+\ln f(x(t)) ]dt$,
\end{center}
характеризующего агрегированную удельную скорость роста потребления на единицу рабочей силы. Таким образом, в терминах фазовой переменной $x$ задача оптимального управления($P_{\epsilon}$), $0<\epsilon<1$,  переписывается в виде следующей задачи ($\tilde{P_{\epsilon}}$):

  $$\dot x(t)=u(t)f(x(t))-\mu x(t), u(t) \in U_{\epsilon}=[0,1-\epsilon],$$
  $$x(0)=x_0,$$
  $$J(x,u)=\int_{0}^{\infty}e^{-pt}[ \ln (1-u(t))+\ln f(x(t)) ]dt \to \max.$$


Здесь $x_0 \in \tilde{G}=(0,\infty)$.

\section{Аналитическое решение задачи, магистрали}\label{2sec:solution}
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
Неоклассическая модель оптимального экономического роста c логарифмической функцией мгновенной полезности формулируется как следующая задача оптимального управления $(P_{\epsilon}), 0<\epsilon<1$ (см. предыдущий пункт):

\begin{equation}\label{2.14}
  \dot K(t)=u(t)F(K(t),L(t)), u(t)\in U_{\epsilon}=[0, 1-\epsilon ],
\end{equation}
\begin{equation}\label{2.15}
  \dot L(t)=\mu L(t),
\end{equation}
\begin{equation}\label{2.16}
  K(0)=K_0, L(0)=L_0,
\end{equation}
\begin{equation}\label{2.17}
  J(K,L,u)=\int_{0}^{\infty}e^{-pt} [\ln(1-u(t))+lnF(K(t),L(t)) ]dt \to \max.
\end{equation}



Здесь $K\in \mathbb{R}^1$ и $L \in \mathbb{R}^1$ --- фазовые переменные (капитал и рабочая сила); $\mu > 0; \rho > 0; K_0 > 0, L_0 > 0$ --- заданные начальные состояния системы; $u$ — управление, характеризующее долю $u(t)F(K(t),L(t))$ произведенного продукта, инвестируемую в основные производственные фонды (капитал) в единицу времени, следующую за моментом времени $t \ge 0$; $F$ — производственная функция. Максимум в задаче $(P_{\epsilon})$  ищется в классе измеримых функций $u \colon [0, \infty)\to U_{\epsilon}$.

 Функция мгновенной полезности
\begin{center}
  $g(K,L,u) = \ln(1-u)+\ln F(K,L), K > 0, L > 0, u \in U_{\epsilon},$
\end{center}
вогнутая по переменной $u$.

Вариант принципа максимума Понтрягина для задачи $(P_{\epsilon})$.

\paragraph{Теорема 2.1.} Пусть $u_*$ — оптимальное управление в задаче ($(P_{\epsilon})$ и $(K_*,L_*)$ — соответствующая оптимальная траектория. Тогда существует такая сопряженная переменная $\psi = (\psi^1, \psi^2)$ (соответствующая тройке $(K_*,L_*,u_*))$, что выполняются следующие условия:
\begin{enumerate}
  \item  оптимальная тройка $(K_*,L_*,u_*)$ вместе с сопряженной переменной $\psi = (\psi^1, \psi^2)$ удовлетворяют на бесконечном полуинтервале времени $[0, \infty)$ основным соотношениям принципа максимума Понтрягина в нормальной форме:

      $$\dot \psi^1(t)=-u_*(t)\frac{\partial F(K_*(t), L_*(t))}{\partial K} \psi^1(t)-$$

\begin{equation}\label{2.18}
-\frac{e^{-\rho t}}{F(K_*(t),L_*(t))} \frac{\partial F(K_*(t),L_*(t))}{\partial K},
\end{equation}

$$\dot \psi^2(t)=-u_*(t)\frac{\partial F(K_*(t), L_*(t))}{\partial L} \psi^1(t)-\mu \psi^2(t)-$$

\begin{equation}\label{2.19}
-\frac{e^{-\rho t}}{F(K_*(t),L_*(t))} \frac{\partial F(K_*(t),L_*(t))}{\partial L},
\end{equation}


\begin{equation}\label{2.20}
  \mathcal{H}(K_*(t),L_*(t),t,u_*(t),\psi (t))=H(K_*(t),L_*(t),t,\psi(t));
\end{equation}

  \item для любого $t \ge 0$ имеем
  \begin{equation}\label{2.21}
    \psi^1(t)>0, \psi^2(t)>0;
  \end{equation}

  \item если существует такое число $\theta>0$, что начиная с некоторого момента времени $\tau \ge 0$ выполняется неравенство

      \begin{equation}\label{2.22}
        u_*(t)\ge \theta,
      \end{equation}
      то выполняется условие трансверсальности

      \begin{equation}\label{2.23}
        \lim_{t\to \infty}\psi^1(t)K_*(t)=0.
      \end{equation}

      Здесь

        $$\mathcal{H}(K,L,t,u,\psi)=uF(K,L)\psi^1+\mu L\psi^2+e^{-\rho t}[ln(1-u)+\ln F(K,L)],$$

        $$H(K,L,t,\psi)=\max_{u\in U_{\epsilon}}\mathcal{H}(K,L,t,u,\psi)$$
      --- функция Гамильтона–Понтрягина и гамильтониан в нормальной форме.
\end{enumerate}


Перейдя в задаче $(P_{\epsilon})$ к фазовой переменной  $x(t)=K(t)$/$L(t)$, $t\ge 0$, получим следующую эквивалентную ей задачу $(\tilde P_{\epsilon}), 0<\epsilon<1$ (см. предыдущий пункт):

\begin{equation}\label{2.24}
  \dot x(t)=u(t)f(x(t))-\mu x(t), u(t) \in U_{\epsilon}=[0,1-\epsilon],
\end{equation}
\begin{equation}\label{2.25}
x(0)=x_0
\end{equation}
\begin{equation}\label{2.26}
J(x,u)=\int_{0}^{\infty}e^{-pt}[ \ln (1-u(t))+\ln f(x(t)) ]dt \to \max.
\end{equation}

Здесь $x_0 \in \tilde{G}=(0,\infty)$ и $f(x)=F(x,1)$ для любого $x\in \tilde{G}$

Следующий вариант принципа максимума Понтрягина для задачи $\tilde{P_{\epsilon}}$ в терминах текущеё сопряённой переменной $p$, текущей функции Гамильтона-Понтрягина $\mathcal{M}$ и текущего гамильтониана $M$.

\paragraph{Теорема 2.2.} Пусть $u_*$ --- оптимальное управление в задаче $\tilde{P_{\epsilon}}$ и $x_*$ — соответствующая оптимальная траектория. Тогда существует такая текущая сопряженная переменная $p$ (соответствующая паре $(x_*,u_*))$, что выполняются следующие условия:
\begin{enumerate}
  \item  оптимальная пара $(x_*,u_*)$ вместе с текущей сопряженной переменной p удовлетворяют на бесконечном полуинтервале времени $[0,\infty)$ основным соотношениям принципа максимума Понтрягина в нормальной форме:

      \begin{equation}\label{2.27}
        \dot p(t)=[p+\mu-u_*(t)\frac{d}{dx}f(x_*(t))]p-\frac{1}{f(x_*(t))} \frac{d}{dx}f(x_*(t)),
      \end{equation}

      \begin{equation}\label{2.28}
        \mathcal{M}(x_*(t),u_*(t),p(t)) = M(x_*(t),p(t)).
      \end{equation}

  \item для любого $t \ge 0$ имеем
  \begin{equation}\label{2.29}
    p(t)>0;
  \end{equation}

  \item если существует такое число $\theta>0$, что начиная с некоторого момента времени $\tau > 0 $ выполняется неравенство
      \begin{equation}\label{2.30}
        u_*(t)\ge \theta,
      \end{equation}
      то выполняется условие трансверсальности
      \begin{equation}\label{2.31}
        \lim_{t\to \infty} e^{\rho t}p(t)x_*(t)=0.
      \end{equation}

      Здесь текущие функция Гамильтона–Понтрягина $\mathcal{M}$ и гамильтониан $M$ для задачи $(\tilde{P_{\epsilon}})$ в нормальной форме определяются стандартным образом:
      \begin{center}
        $\mathcal M(x,u,p)=(uf(x)-\mu x)p +\ln(1-u)+\ln f(x),$
      \end{center}

      \begin{equation}\label{2.32}
        M(x,p) = \max_{u\in U_{\epsilon}}\mathcal M(x,u,p).
      \end{equation}

\end{enumerate}

Ниже нам потребуется следующий вспомогательный результат.

\paragraph{Лемма 2.1.} Пусть $\phi_1\colon \tilde G \mapsto \mathbb R^1$ и $\phi_2\colon \tilde G \mapsto \mathbb R^1$ --- две такие вогнутые функции, что
\begin{equation}\label{2.33}
  \phi_1(x)\ge \phi_2(x) \text{ для любого }x\in \tilde G.
\end{equation}

Пусть, кроме того, существует такая точка $\xi \in \tilde G$, что
\begin{equation}\label{2.34}
  \phi_1(\xi)=\phi_2(\xi).
\end{equation}

Тогда функции $\eta_1, \eta_2\colon \tilde G\mapsto \mathbb R^1$, определённые равенствами

\begin{equation}\label{2.35}
  \eta_1(x)=
  \begin{cases}
            \phi_1(x), & \text{если $x\le\xi$,} \\
            \phi_2(x), & \text{если $x>\xi$,}
          \end{cases}
          \eta_2(x)=
          \begin{cases}
            \phi_2(x), & \text{если $x\le\xi$,} \\
            \phi_1(x), & \text{если $x>\xi$}
          \end{cases}
\end{equation}
являются вогнутыми.

\paragraph{Лемма 2.2.} Для любого $p>0$ текущий гамильтониан $M$ задачи $(\tilde P_{\epsilon})$ является вогнутой функцией фазовой переменной $x$ на множестве $\tilde G$.
Из этого вытекают 2 результата

\paragraph{Следствие 2.1.}  Пусть допустимая пара $(x_*,u_*)$ вместе с текущей сопряженной переменной p удовлетворяют условиям принципа максимума Понтрягина в нормальной форме для задачи $(\tilde P_{\epsilon})$ и существует такое число $\theta>0$, что начиная с некоторого момента времени $\tau \ge 0$ выполняется неравенство $u_*(t) \ge \theta$. Тогда пара $(x_*,u_*)$ оптимальная в задаче $\tilde P_{\epsilon}$.

\paragraph{Следствие 2.2.} Пусть для допустимой пары $(x_*,u_*)$ существует такое число $\theta>0$, что начиная с некоторого момента времени $\tau \ge 0 $ выполняется неравенство $u_*(t) \ge \theta$. Тогда для оптимальности пары $(x_*,u_*)$ в задаче $(\tilde P_{\epsilon})$ необходимо и достаточно, чтобы вместе с некоторой текущей сопряженной переменной $p$ она удовлетворяла условиям принципа максимума Понтрягина в нормальной форме.

Любая пара $(x_*,p_*)$ принимает значение в
\begin{center}
  $\Gamma =\{(x,p)\in \mathbb R^2\colon x>0, p>0 \}$.
\end{center}
Максимум в (\ref{2.32}) достигается в точке
\begin{equation}\label{2.36}
  u(x,p)=
  \begin{cases}
    0, & \text{если $0<p<\frac{1}{f(x)},$} \\
    1-\frac{1}{f(x)p}, & \text{если $\frac{1}{f(x)}\le p\le\frac{1}{\epsilon f(x)},$} \\
    1-\epsilon, & \text{если $p>\frac{1}{\epsilon f(x)}.$}
  \end{cases}
\end{equation}
и всюду в множестве $\Gamma$ имеем
\begin{center}
  $M(x,p)=(u(x,p)f(x)-\mu x)p + \ln(1-u(x,p))+\ln f(x).$
\end{center}

Определим множества $\Gamma_1, \Gamma_2, \Gamma_3$:
\begin{center}
  $\Gamma_1=\{(x,p)\in\Gamma\colon =0<p<\frac{1}{f(x)}\}$,

   $\Gamma_2=\{(x,p)\in\Gamma\colon \frac{1}{f(x)}<p<\frac{1}{\epsilon f(x)}\}$,

    $\Gamma_3=\{(x,p)\in\Gamma\colon p>\frac{1}{\epsilon f(x)}\}$.
\end{center}
$\Gamma=\Gamma_1 \bigcup \Gamma_2 \bigcup \Gamma_3.$

Теперь мы рассматриваем поведение гамильтоновой системы принципа максимума в каждом из множеств. После этого мы можем сделать некоторые выводы.

В силу проведенного анализа необходимым условиям оптимальности (теорема 2.2) для любого начального состояния $x_0$ удовлетворяет единственная траектория гамильтоновой системы принципа максимума. В зависимости от начального условия $x_0$ это либо правая равновесная траектория, либо левая равновесная траектория, либо стационарная траектория $(\hat x,\hat p)$. Все остальные траектории гамильтоновой системы не удовлетворяют условиям теоремы 2.2, поскольку либо у них на конечном интервале времени координата p обращается в нуль и они покидают множество $\Gamma$, либо на некотором интервале времени они имеют направление движения, противоположное направлению движения левой равновесной или правой равновесной траектории (см. рис.), либо они асимптотически приближаются при $t \to \infty$ к прямой $V_{32}^0$ (или движутся по ней) и при этом нарушается условие трансверсальности (2.19).

Таким образом, необходимые условия оптимальности (теорема 2.2) однозначно характеризуют оптимальное управление в задаче $(\tilde{P}_{\epsilon})$  в терминах левой и правой равновесных траекторий $p_1(x)$ и $p_2(x)$. В случае, когда левая равновесная траектория пересекает кривую $V_{\epsilon}$ в единственной точке $(\hat x_{\epsilon}, \hat p_{\epsilon})$, а правая равновесная траектория пересекает кривую $V$ в единственной точке $(\tilde x,\tilde p)$, мы имеем
\begin{center}
  $u_*(x)$=$
  \begin{cases}
    1-\epsilon, & \text{при $x\in (0,\hat x_{\epsilon})$}, \\
    1-\frac{1}{p_1(x)f(x)}, & \text{при $x\in [\hat x_{\epsilon},\hat x)$}, \\
    1-\frac{1}{f(\hat x)\hat p}, & \text{при $x=\hat x$}, \\
    1-\frac{1}{p_2(x)f(x)}, & \text{при $x\in (\hat x, \tilde x)$}, \\
    0, & \text{при $x\in [\tilde x,\infty)$}.
  \end{cases}$
\end{center}
Для произвольного начального состояния $x_0 > 0$ оптимальный синтез $u_*(x), x \in(0,\infty)$, однозначно определяет оптимальную траекторию $x_*$ в задаче $(\tilde P_{\epsilon})$ как решение задачи Коши
\begin{center}
  $x(t)=u_*(x(t))f(x(t))-\mu x(t), x(0) = x_0$
\end{center}
и соответствующее оптимальное управление $u_*$ как функцию $u_*(t)=u_*(x_*(t)),t \in [0,\infty)$.
\subsection{Магистрали}
Основы магистральной теории были заложены в тридцатые года прошлого века Джоном Нейманом. В магистральной теории изучается модель расширяющейся экономики.

Магистраль – это траектория пропорционального сбалансированного роста экономики с технологическим (максимальным) темпом $ x* $. Начальное $ L_{*0} $ и конечное $ L_{*t} $ состояния экономики при отыскании магистрали не задают. Пропорциональный рост означает, что соотношение (пропорции) между объемами затрат или выпуска разных видов продукции в разные элементарные отрезки времени не изменяется. Магистральная теория позволяет вычислять оптимальные траектории экономического роста. Доказано, что оптимальный путь проходит
хотя бы частично по магистральной траектории. Эта теория дала
основу для создания прикладных методов решения задач оптимального экономического роста.

Магистральная теория строится на основе следующей аксиоматики.
\begin{enumerate}
  \item Аксиома невозможности производства. При отсутствии затрат производство невозможно.
  \item Аксиома преобразуемости. При определенных технологических способах любой набор затрат можно преобразовать в другой набор выпуска.
  \item Аксиома продуктивности Все продукты можно произвести при помощи технологий.
\end{enumerate}
Пусть $ u_* $ — оптимальное управление в задаче $ (P_\varepsilon) $ (или, что то же самое, в задаче $ (\tilde{P}_\varepsilon) $), $ (K_*,L_*) $ — соответствующая оптимальная траектория. Тогда $ x_*(t)=K_*(t)/L_*(t), t \ge 0 $, есть соответствующая $ u_* $ оптимальная допустимая траектория в задаче $ (\tilde{P}_\varepsilon) $).

$ x_*(t) = x_{*}^{1}, x_{*}^{2}, ... , x_{*}^{m} $ - множество оптимальных траекторий. Рост экономики ограничен наиболее сдерживающим темпом,  который определяется показателем
\begin{center}
	$ \bar{x} = \underset{i = 1,m}{\min} x_{*}^{i}. $
\end{center}
Показатель $ \bar{x} $
является условно минимальным темпом роста производства. Условность состоит в том, что из множества Z применяется одна из существующих групп технологий $ Z^r \in Z, r = 1, 2, …, s. $ Значение показателя $ \bar{x} $ может быть увеличено применением других групп технологий из множества $ z $. Тогда оптимизация выполняется по Парето. Критерием оптимизации становится достижение равенства
\begin{center}
	$ x* = \underset{r=1,s}{\max} \underset{i=1,m}{\min} x_{*}^{i}. $
\end{center}

Показатель $ x* $
называется технологическим темпом роста. В магистральной теории доказано, что с таким темпом происходит максимально возможный рост производства продукции.
Магистраль же строится по следующей формуле:
\begin{center}
	$ L*_{t} = x* L*_{t-1}. $
\end{center}

Магистраль, которая проходит через точки $ L*_0 $ и $ L*_t $, как правило не проходит через точку фактического начала $ L_*(0) $ и желаемого конечного состояния экономики $ L_*(t) $.

Поэтому наилучшая стратегия экономического роста состоит в разделении реальной траектории на три участка. В начальном участке экономика из точки фактического состояния $ L_*(0)$ выходит на магистраль, затем она движется по магистрали и на третьем участке выходит в заданную точку состояния экономики $ L_*(t) $. В частных случаях участки могут быть нулевыми. Магистраль не зависит от горизонта планирования (числа элементарных отрезков времени).

\section{Применение методов EMPC}\label{2sec:useempc} 
%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%%
Существует 3 подхода в решении задач через EMPC [3]

\begin{enumerate}
  \item Базовый EMPC без дополнительных ограничений.
  \item EMPC с терминальным ограничением равенства.
  \item Подход, в котором критерий качества дополняется специально выбранной терминальной стоимостью.
\end{enumerate}

Задача оптимального управления без дополнительных ограничений и терминальной стоимости, применяемая в ЕМРС без дополнительных ограничений, имеет вид

$$ I_1(z,u)=\int_{\tau}^{\tau+T}e^{-\rho t}[\ln(1-u(t))+\ln(f(x(t)))]dt \to \max,$$
$$\dot z(t)=u(t)f(z(t))-\mu x(t),$$
$$x(\tau)=\frac{K(\tau)}{L(\tau)},$$
$$u(t)\in[0,b-\epsilon], t\in[\tau, \tau + T].$$

Если дополнить задачу терминальным ограничением равенства то она примет вид

$$ I_2(z,u)=\int_{\tau}^{\tau+T}e^{-\rho t}[\ln(1-u(t))+\ln(f(x(t)))]dt \to \max,$$
$$\dot z(t)=u(t)f(z(t))-\mu x(t),$$
$$x(\tau)=\frac{K(\tau)}{L(\tau)}, z(\tau + T)=z_1,$$
$$u(t)\in[0,b-\epsilon], t\in[\tau, \tau + T].$$
Требование $z(\tau + T)=z_1,$ означает, что в момент $\tau+T$ $z$ должно равняться магистральному значению.

Если дополнить критерий качества задачи специально подобранной линейной терминальной стоимостью то задача примет вид

$$ I_3(z,u)=\alpha e^{-\rho(\tau+T)}z(\tau+T)+\int_{\tau}^{\tau+T}e^{-\rho t}[\ln(1-u(t))+\ln(f(x(t)))]dt \to \max,$$
$$\dot z(t)=u(t)f(z(t))-\mu x(t),$$
$$x(\tau)=\frac{K(\tau)}{L(\tau)},$$
$$u(t)\in[0,b-\epsilon], t\in[\tau, \tau + T].$$
где $\alpha$ --- весовой коэффициент

\chapter{Численные эксперименты}\label{chap3}
Продолжим рассматривать задачу

\begin{equation}
  \dot x(t)=u(t)f(x(t))-\mu x(t), u(t) \in U_{\epsilon}=[0,1-\epsilon],
\end{equation}
\begin{equation}
x(0)=x_0
\end{equation}
\begin{equation}
J(x,u)=\int_{0}^{\infty}e^{-pt}[ \ln (1-u(t))+\ln f(x(t)) ]dt \to \max
\end{equation}

Решение будем искать в классе кусочно-постоянных функций. Это значит, что $u$ примет вид
$$u(t)=u_n, t\in[nh, (n+1)h], n=0,1\dots,N-1$$
$N$ --- горизонт планирования, $h$ --- период дискретизации.

Тогда можем дискретизировать задачу. Получим такой вид

$$J(x,u)=\sum_{0}^{T-1}\frac{\ln(1-u_i)+\ln(f(x_i))}{-\rho}(e^{-\rho(i+1)h}-e^{-\rho i h}),$$
$$u_i(t)\in U=[0, 1-\epsilon], x(0)=x_0,$$
$$x_{i+1}=u_i f(x_i)-\mu x_i, i=0,1,\dots,N-1$$

Задача оптимального управления в классе дискретных управлений свелась к задаче нелинейного программирования. В matlab существуют встроенные процедуры для решения таких задач.

Для численного эксперимента примем $T=10$, $f(x)=x^{\alpha}$, $\alpha=0.3$, $\mu=0.03$, $\rho=0.05$, $\epsilon=0.01$.

\paragraph{Код}

\begin{verbatim}
global Horizon
Horizon=50;
iterations = 50;
x0 = 5;

[X, U] = solve(iterations, x0, 0, 0.3*ones(Horizon,1));
disp(X(1,:));
disp(U(1,:));

plot((0:iterations),X(1,:), '-k', 'LineWidth',3);
%plot((0:iterations-1),U(1,:), '-r', 'LineWidth',3);

function [x, u] = solve(iterations, x0, t0, u0)
t = t0;
global Horizon;
x = zeros(Horizon + 1,iterations);
x(1,1) = x0;
u = zeros(Horizon,iterations);
for i = 1:iterations
u(:, i) = fmincon(@(u) criteria(x(1,i),t,u),u0,[],...
    [],[],[],zeros(Horizon,1),ones(Horizon,1)-0.01);
for j = 1:Horizon
x(j+1,i) = nextX(t + j,x(j,i),u(j,i));
end
x(1,i+1) = x(2,i);
t = t + 1;
end
end

function x = nextX(t,k,c)
[~, x] = ode45(@(t,k) deq(k,c),[t, t+1], k);
x = x(end,:);
end

function l = stagecost(t, x, u)
l = (log(1 - u) + log(x^0.3))*(exp(-0.05 * (t+1))...
    -exp(-0.05 * t))/0.05;
end

function cost = criteria(x0,t0,u)
cost = 0;
x = x0;
t = t0;
global Horizon;
for i = 1:Horizon
cost = cost + stagecost(t, x, u(i));
x = nextX(t,x,u(i));
t = t + 1;
end
end

function dx = deq(x,u)
dx = u*x^(0.3) - 0.03*x;
end
\end{verbatim}
